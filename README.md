# ブラックチャンネル キャラクターDB

ブラックチャンネルのキャラクター情報を素早く確認できる、シンプルでスタイリッシュなWEBサイトです。

## ディレクトリ構成

*   `index.html` : サイトのメイン構造（HTML）
*   `style.css` : サイトのデザイン（CSS。ダークテーマ＆グリッチ感のあるプレミアムスタイル）
*   `app.js` : キャラクター情報の読み込み、検索、身長チャート生成ロジック
*   `characters.json` : キャラクターのデータ（JSON形式）

## ローカルでの起動方法

CORS（セキュリティ制限）のため、ローカルでファイルをダブルクリックで開いた（`file://` スキームの）状態だと `characters.json` からデータを直接 fetch できない場合があります。（その場合も、`app.js` 内に定義したフォールバック用の初期データが自動で表示される設計になっています）

JSONファイルを更新して動的に読み込ませるには、以下のいずれかの方法で簡易ローカルサーバーを起動してください。

### 方法1：Pythonを使用する場合（おすすめ）
PowerShell やターミナルを開き、本ディレクトリで以下のコマンドを実行します。
```bash
python -m http.server 8000
```
起動後、ブラウザで `http://localhost:8000` にアクセスします。

### 方法2：Node.js / npm の serve パッケージを使用する場合
```bash
npx serve .
```
起動後、ターミナルに表示されるアドレス（通常は `http://localhost:3000` など）にアクセスします。

## キャラクターデータの更新・追加方法

`characters.json` を任意のテキストエディタで直接編集するだけで、サイト上のキャラクター情報が自動で更新されます。

### JSONのデータ構造例
```json
  {
    "name": "キャラ名",
    "height": 170,  // 数値。不明な場合は "?"
    "birthday": "MM-DD",  // "09-06" のような形式。不明な場合は "?"
    "firstPerson": "一人称",  // 不明な場合は "?"
    "catchphrase": "口癖"  // 不明な場合は "?"
  }
```

*   **身長 (height)** を数値で登録すると、「Height Comparison Chart (身長比較)」のグラフに自動で反映されます。
*   新しいキャラクターを追加したい場合は、JSON配列の末尾に要素を追加してください。
